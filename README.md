# Space Dodge

A browser-based side-scrolling space game. Dodge asteroids, survive the cave gauntlet, and dodge past the boss.

## Controls

| Key | Action |
|-----|--------|
| `W` / `↑` | Move up |
| `S` / `↓` | Move down |
| `A` / `←` | Move left |
| `D` / `→` | Move right |

- **Boss fight** triggers at 65 seconds of gameplay
- **Cave gauntlet** begins at score 256 — wall turrets fire into the gap
- Mobile: tap screen quadrants to move

## How to run

```bash
npx serve .
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.
